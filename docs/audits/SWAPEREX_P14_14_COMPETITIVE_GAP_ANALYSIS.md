## Priority tiers

### Core requirements (address first)

1. Network tier clarity (swap vs view-only)
2. Gas / total cost visibility
3. Deep-linkable main tabs
4. Post-swap receipt + explorer link

### High-value improvements

1. Persistent transaction activity
2. Public status/support page
3. Mobile WC validation
4. og:image + brand consistency

### Growth features

1. Expanded audited pairs
2. Screener → swap prefill polish
3. Radar integration in swap warnings

### Advanced trading

1. Limit orders
2. Cross-chain
3. MEV protection
4. Additional L2 swap chains

### Low-priority extras

1. Referral program
2. Social login (intentionally off)
3. Injected wallet support (intentionally off)

---

## Strategic position

Swaperex is a **focused commission DEX interface**, not a full aggregator at Uniswap scale. Competitive gaps in breadth are **expected**; gaps in **clarity and trust** are the priority fix areas.
